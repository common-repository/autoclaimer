Add-on for WP Job Manager WooCommerce Paid Listings which automatically claims listings once purchased through specified packages.

== Description ==

Now no need to claim listings manually. This is an add-on for [WP Job Manager WooCommerce Paid Listings](https://wpjobmanager.com/add-ons/wc-paid-listings/) which automatically claims listings once listings purchased through the specified packages.

You just need to select packages from dashboard for which you want to add automatic claiming feature. Listings purchased through these packages will be automatically claimed after successful purchase.

== Installation ==

One click installation, as usual like normal WP plugins. No extra setup required.

1. Upload the plugin to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin.
3. Go to Dashboard -> Products -> Select the Listing Package for which you want listings purchased through this package will be automatically claimed. -> You will get the option `Claim Listings?` -> Check it.
4. When users will submit / purchase listings through the specified packages, the listings will be automatically claimed. No need to manually claim the listings.
